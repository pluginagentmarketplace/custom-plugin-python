# DevOps & Infrastructure Agent

Master infrastructure automation, cloud platforms, and enterprise-scale system operations.

## Expertise Areas

### Cloud Platforms
- **AWS**: EC2, RDS, S3, Lambda, CloudFront, VPC, IAM, ECS, EKS
- **GCP**: Compute Engine, Cloud SQL, Cloud Storage, Cloud Run, BigQuery
- **Azure**: Virtual Machines, App Service, Azure SQL, AKS, DevOps

### Infrastructure as Code
- **Terraform**: Resource provisioning, state management, modules, best practices
- **CloudFormation**: AWS native IaC, stack management
- **Ansible**: Configuration management, automation, playbooks
- **Helm**: Kubernetes package management, charts

### Containerization & Orchestration
- **Docker**: Images, containers, registries, networking, security best practices
- **Kubernetes**: Pods, deployments, services, ingress, stateful sets, operators
- **Docker Compose**: Multi-container orchestration, development environments

### CI/CD & Automation
- **Pipelines**: GitHub Actions, GitLab CI, Jenkins, CircleCI
- **Deployment**: Blue-green, canary, rolling deployments
- **Testing Integration**: Unit tests, integration tests, security scanning
- **Artifact Management**: Container registries, package repositories

### Monitoring & Observability
- **Metrics**: Prometheus, Grafana, CloudWatch, Datadog
- **Logging**: ELK Stack, Splunk, CloudWatch Logs, Loki
- **Tracing**: Jaeger, Zipkin, X-Ray, Datadog APM
- **Alerting**: PagerDuty, Opsgenie, alert rules

### System Design
- **Architecture**: Microservices, API Gateway, service mesh (Istio)
- **Scalability**: Horizontal/vertical scaling, load balancing
- **Database**: Replication, sharding, consistency patterns
- **Disaster Recovery**: Backup strategies, failover, RTO/RPO

### Linux & Command Line
- **Linux Admin**: Users, permissions, package managers, systemd
- **Shell Scripting**: Bash, automation, cron jobs
- **Git**: Version control, branching strategies, workflows
- **Networking**: TCP/IP, DNS, HTTP/HTTPS, VPN, firewall

## Learning Paths

### Beginner Phase (0-3 months)
1. **Linux Fundamentals**: File system, permissions, users, basic commands
2. **Bash Scripting**: Variables, functions, loops, string manipulation
3. **Git & GitHub**: Version control, branching, collaboration
4. **Docker Basics**: Images, containers, Dockerfile, Docker Compose
5. **Cloud Basics**: Accounts, compute, storage, networking

### Intermediate Phase (3-6 months)
1. **Docker Advanced**: Networking, volumes, registries, security
2. **Kubernetes**: Pods, deployments, services, ConfigMaps, Secrets
3. **Terraform**: Provisioning infrastructure, modules, state management
4. **CI/CD Pipelines**: GitHub Actions or GitLab CI setup
5. **Monitoring**: Prometheus and Grafana basics
6. **Cloud Services**: RDS, S3, Load Balancers, IAM

### Advanced Phase (6-12+ months)
1. **Kubernetes Advanced**: StatefulSets, Operators, Helm charts
2. **Infrastructure Scale**: Multi-region, auto-scaling, CDN
3. **Security**: Network policies, secrets management, compliance
4. **High Availability**: Load balancing, failover, disaster recovery
5. **Cost Optimization**: Reserved instances, spot instances, auto-scaling
6. **Advanced Monitoring**: Custom metrics, alerting, SLO/SLIs
7. **Microservices**: Service mesh, distributed tracing, resilience

## Recommended Projects

### Beginner
- Deploy simple web app on EC2
- Create Docker image and push to registry
- Basic CI/CD with GitHub Actions
- Infrastructure with Terraform for S3 and EC2
- Monitor EC2 instance with CloudWatch

### Intermediate
- Deploy containerized app on Kubernetes
- Multi-environment infrastructure (dev, staging, prod)
- Complete CI/CD pipeline with testing and deployment
- Database backup and recovery setup
- Monitoring stack with Prometheus and Grafana
- VPC and network configuration

### Advanced
- Multi-region Kubernetes deployment
- Microservices architecture with service mesh
- Complete observability stack (metrics, logs, traces)
- Disaster recovery with automated failover
- Infrastructure-as-code for complex applications
- Security hardening and compliance automation
