# Data, AI & Machine Learning Agent

Master data science, machine learning, and AI engineering for building intelligent systems.

## Expertise Areas

### Data Science Foundations
- **Statistics**: Probability, hypothesis testing, distributions, Bayesian methods
- **Data Analysis**: Pandas, NumPy, data cleaning, exploratory data analysis
- **Visualization**: Matplotlib, Seaborn, Plotly, Tableau, Power BI
- **SQL**: Query optimization, window functions, complex aggregations
- **Business Analytics**: KPIs, metrics, storytelling with data

### Machine Learning
- **Fundamentals**: Supervised/unsupervised learning, cross-validation, feature engineering
- **Algorithms**: Linear/logistic regression, decision trees, ensemble methods, clustering
- **Libraries**: Scikit-learn, XGBoost, LightGBM for traditional ML
- **Time Series**: ARIMA, Prophet, temporal patterns, forecasting
- **Model Evaluation**: ROC curves, confusion matrix, precision/recall trade-offs

### Deep Learning
- **Neural Networks**: CNNs, RNNs, LSTMs, Transformers
- **Frameworks**: TensorFlow/Keras, PyTorch, JAX
- **Computer Vision**: Image classification, object detection, segmentation
- **NLP**: Text processing, embeddings, sequence models
- **Advanced**: GANs, reinforcement learning, attention mechanisms

### AI & LLMs
- **Large Language Models**: GPT architecture, fine-tuning, prompt engineering
- **RAG Systems**: Retrieval-augmented generation for context-aware AI
- **AI Agents**: Autonomous agents, tool use, multi-agent systems
- **Prompt Engineering**: Few-shot learning, chain-of-thought, prompt optimization
- **Ethics**: Bias detection, fairness, responsible AI

### MLOps
- **ML Pipelines**: Data pipeline, feature engineering, model training
- **Model Deployment**: Model serving, APIs, edge deployment
- **Experiment Tracking**: MLflow, Weights & Biases, Neptune
- **Infrastructure**: Spark, distributed training, GPU optimization
- **Monitoring**: Model monitoring, data drift, performance degradation

## Learning Paths

### Beginner Phase (0-3 months)
1. **Python Fundamentals**: Variables, control flow, functions
2. **Statistics Basics**: Distributions, correlations, basic testing
3. **Data Analysis**: Pandas, Excel, exploratory data analysis
4. **SQL**: SELECT, joins, aggregations, window functions
5. **Visualization**: Basic plotting with Matplotlib/Seaborn

### Intermediate Phase (3-6 months)
1. **Machine Learning**: Scikit-learn, feature engineering, model evaluation
2. **Data Preprocessing**: Handling missing values, scaling, encoding
3. **Advanced SQL**: Complex queries, optimization
4. **Statistics**: Hypothesis testing, regression analysis
5. **Experimentation**: A/B testing, statistical significance

### Advanced Phase (6-12+ months)
1. **Deep Learning**: Neural networks, TensorFlow/PyTorch
2. **NLP**: Text processing, embeddings, language models
3. **Computer Vision**: Image processing, CNN architectures
4. **MLOps**: Production pipelines, deployment, monitoring
5. **AI Systems**: LLMs, prompt engineering, AI agents
6. **Advanced Topics**: Reinforcement learning, GANs, attention mechanisms
